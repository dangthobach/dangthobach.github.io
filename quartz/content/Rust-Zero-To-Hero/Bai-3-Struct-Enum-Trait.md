# Bài 3: Structs, Enums & Traits - "Class" kiểu mới trong Rust

Chào Chuyên gia Java, ở bài này chúng ta sẽ trả lời câu hỏi: "Nếu không có Class và Interface, làm sao tôi đóng gói (encapsulate) và trừu tượng hóa (abstract) dữ liệu?"

## 1. Structs: Không chỉ là POJO
Trong Java, bạn dùng Class để giữ dữ liệu. Trong Rust, chúng ta dùng `struct`.

### Java (Class/Record)
```java
public class User {
    private String name;
    private int age;
    // Constructor, Getters, Setters...
}
```

### Rust (Struct)
```rust
struct User {
    name: String,
    age: u8,
}

// Thêm hành vi (Method) cho struct
impl User {
    fn new(name: String, age: u8) -> Self {
        Self { name, age }
    }

    fn say_hello(&self) {
        println!("Chào, tôi là {}!", self.name);
    }
}
```
**Điểm khác biệt:** Rust tách biệt hoàn toàn **Dữ liệu (struct)** và **Hành vi (impl)**. Điều này giúp code cực kỳ tường minh.

## 2. Enums: "Vũ khí hạng nặng" của Rust
Enum trong Java thường chỉ là danh sách hằng số. Enum trong Rust (Algebraic Data Types) mạnh hơn rất nhiều vì mỗi variant có thể chứa dữ liệu khác nhau.

### Ví dụ: Trạng thái đơn hàng
```rust
enum OrderStatus {
    Pending,
    Shipped(String), // Chứa mã vận đơn
    Delivered { warehouse: String, time: u32 }, // Chứa struct ẩn
    Cancelled(String), // Chứa lý do hủy
}

fn process_order(status: OrderStatus) {
    match status {
        OrderStatus::Pending => println!("Đang chờ..."),
        OrderStatus::Shipped(code) => println!("Đang giao, mã: {}", code),
        OrderStatus::Delivered { warehouse, .. } => println!("Đã giao tới {}", warehouse),
        _ => println!("Trạng thái khác"),
    }
}
```

## 3. Traits: Interface nhưng "xịn" hơn
Trait chính là Interface trong Java, nhưng nó không có khái niệm kế thừa (Inheritance). Nó định nghĩa các hành vi chung.

### Ví dụ: Trait Summary
```rust
trait Summary {
    fn summarize(&self) -> String;
    
    // Default method
    fn author(&self) -> String {
        String::from("Unknown")
    }
}

impl Summary for User {
    fn summarize(&self) -> String {
        format!("User {} ({} tuổi)", self.name, self.age)
    }
}
```

## 4. Xử lý lỗi: Tạm biệt NullPointerException & Exceptions
Rust dùng Enum: `Option<T>` và `Result<T, E>`.

*   **Option<T>**: Thay thế cho `null`.
*   **Result<T, E>**: Thay thế cho `Exception`.
