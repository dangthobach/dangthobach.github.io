# 📥 Inbox

Mọi note mới đều vào đây trước. Xử lý định kỳ (cuối ngày hoặc cuối tuần):

1. Đặt tên rõ ràng
2. Thêm tags phù hợp
3. Tạo wikilinks đến concepts liên quan
4. Move sang `concepts/`, `patterns/`, hoặc `projects/`
5. Cập nhật MOC tương ứng

> **Quy tắc:** Inbox không bao giờ là nơi lưu trữ lâu dài.
