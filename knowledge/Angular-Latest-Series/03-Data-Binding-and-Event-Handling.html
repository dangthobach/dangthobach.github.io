Nguyên tắc cốt lõi:

  • [ ] → Dữ liệu đi từ TS → HTML (đọc)
  • ( ) → Dữ liệu đi từ HTML → TS (viết/gọi)
  • [( )] → Hai chiều đồng thời (đọc + viết)
  • {{ }} → Nội suy chuỗi vào DOM

sequenceDiagram
    participant P as Parent
    participant C as Child (model)
